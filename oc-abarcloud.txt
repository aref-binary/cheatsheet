### Common commands  
oc logs -f bc/flight-search  
oc logs -f pod-name
oc get pods  
cat .env.abarcloud | oc set env -e - dc/flight-search  
oc rsh deploymentconfig/flight-search  
oc rsync flight-search-10-xpugq:log/production.log /home/sizief/projects  
  
  
###delete app  
oc delete all -l app=postgresql-beta   
  
  
###delete deployment configs  
oc delete dc postgresql-meta-search -n flight-meta-search  
  
  
#first time app  
add private repo ssh key to abarcloud  
create new app: oc new-app --strategy=docker git@gitlab.com:sizief/flight-search.git  
get build config name: oc get buildconfigs  
link build config to secret: oc patch buildconfig/flight-search -p '{"spec":{"source":{"sourceSecret":{"name":"gitlab-flight-meta-search-key"}}}}'  
build: oc start-build buildconfig/flight-search   
add env variables: cat .env.production | oc set env -e - dc/flight-search   
start new pod: oc rollout latest flight-search  
oc expose service/flight-search  
oc get routes  
  
  
### update the app  
oc start-build buildconfig/flight-search  
oc logs -f buildconfig/flight-search  
oc get deploymentconfigs  
it is automat but in case you want the mannual command: oc rollout latest <my-deployment-config>
